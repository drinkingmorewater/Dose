## 🚀 快速开始

```bash
cd yiwei-dose-v1.5
python -m http.server 8000
open http://localhost:8000
```

**操作流程**：
1. **开方页（首页）**：输入复杂情绪文本 → 点「抓药」
2. **AI 把脉浮层**：4 步加载 → 三段分析（情绪拆解/成因/建议）→ 双模式选方 → 取方
3. **音方笺**：三段 Tab —— 处方（扩容详解）/ 剧场（呼吸环+播放）/ 合音（衍生即兴）
4. **合音工作台**：进阶玩家选 2-3 件乐器自由编曲
5. **我的页**：登录 + 推荐 + 主题切换

## 🛠️ 技术栈

- **前端**：原生 HTML / CSS / JavaScript（无构建依赖）
- **音频**：Web Audio API + AudioRegistry 双层架构
- **字体**：Google Fonts（ZCOOL XiaoWei + Noto Serif SC + Cormorant Garamond）
- **存储**：localStorage（用户名 + 主题偏好）
- **AI 模拟**：KEYWORD_MAP 关键词识别（决赛接 LLM）
- **响应式**：桌面手机框 + 移动端全屏
