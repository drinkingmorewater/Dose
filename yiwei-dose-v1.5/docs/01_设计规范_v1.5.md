## ✍️ 二、字体革新（v1.5 重点视觉整改）

### 删除字体
- ❌ ZCOOL QingKe HuangYou（魏碑体）—— 棱角突兀、笔画狭窄
- ❌ Ma Shan Zheng（毛笔行书）—— 太随性，与产品端庄感冲突
- ❌ Liu Jian Mao Cao（草书）—— v1.4 已删除

### 新主字体系统
| 层级 | 字体族 | 字号 | 用途 |
|---|---|---|---|
| **全局主字体** | **ZCOOL XiaoWei**（瘦金体风格）| 11-15px | 正文、标签、按钮、提示、滚动标语、所有中文展示区域 |
| **端庄重标题** | **Noto Serif SC 900**（开源版方正小标宋）| 13-24px | 页面标题、品牌名、处方关键字段（脉象/主药/辅药）、底部导航 |
| **英文 / 数字** | Cormorant Garamond / Georgia | 9-15px | 所有英文、阿拉伯数字、品牌 v 号 |

### 字体特性说明
> **ZCOOL XiaoWei** 是开源最接近"复刻西窗烛主页方正舒展瘦金体风格"的字体：
> - 笔画舒展、横细竖粗（瘦金体核心特征）
> - 工整方正、不局促
> - 国风质感强，宋人写意
> - 商用免费

### Google Fonts 加载（v1.5 精简）
```html
<link href="https://fonts.googleapis.com/css2?
  family=Noto+Serif+SC:wght@400;500;700;900&
  family=ZCOOL+XiaoWei&
  display=swap" rel="stylesheet">
```

### 字体规则
1. **全局应用**：导航栏、按钮、标签、正文、处方、弹窗、所有中文 ⇒ `ZCOOL XiaoWei`
2. **重标题**：屏标题、品牌名、处方字段名 ⇒ `Noto Serif SC 900`
3. **英文/数字**：保持 serif 衬线（Cormorant Garamond），不应用 ZCOOL XiaoWei
4. **字间距**：标题 8-14px，正文 1-3px，符合"工整方正"观感

## 🆕 四、v1.5 核心新组件

### 4.1 AI 智能分析浮层（开方页）
- 触发：点击「抓药」按钮
- 加载阶段：4 步循环文案 ——「正在为你把脉…→拆解情绪成因…→匹配音律谱系…→调和五音五脏…」
- 结果三段：情绪拆解 / 成因推演 / 音律建议
- 双模式切换：
  - **智能推送**：AI 直接给出主药+辅药+服法
  - **自主选方**：列出 4 个候选音方，用户手动挑选

### 4.2 音方笺三段式 Tab（核心整合）
| 段 | 内容 | 关键元素 |
|---|---|---|
| 处方 | 脉象 / 主药 / 辅药 / 服法 + **脉象详解 / 药性释义 / 服法详解 / 寄语诗** | 内容大幅扩容 |
| 剧场 | 水墨流水 + 呼吸环 + 频谱 + 播放控件 + 音之来历 | 嵌入式，非独立页 |
| 合音 | 主方 + 6 件相宜衍生乐器，1-3 件即兴合鸣 | 衍生于当前处方，非独立 |

### 4.3 "我的"页（合并卡片）
- 未登录态：朱砂印章 + 入山按钮 + 三方登录 + 游客通道
- 已登录态：头像 + 用户名 + 音方统计 + 退出按钮
- 下方：今日推荐 4 卡 + 时辰贴士 + 主题切换

### 4.4 底部导航（v1.5 精简至 4 项）
```
📜 开方   🎴 音方笺   🎼 合音   🏮 我的
```

## 📊 八、数据模型（v1.5 PrescriptionRule 大幅扩展）

```typescript
interface PrescriptionRule {
  // 基础（v1.0+）
  sub: string;            // 副标题（如「徵音入心 · 安神方」）
  main: string;           // 主药
  aux: string;            // 辅药
  usage: string;          // 服法

  // 🆕 v1.5 扩容

  herbDetail: string;     // 药性释义（约 100 字 · 含<b>主辅药</b>）
  usageDetail: string;    // 服法详解（约 80 字 · 含呼吸节奏）
  deriveInstruments: string[];  // 衍生合音 6 件相宜乐器
  songId: string;         // 剧场曲目 ID
  songName: string;       // 剧场曲名
  songMain: string;       // 主奏乐器 ID（剧场播放用）
  origin: string;         // 音之来历（约 60 字）
}
```
