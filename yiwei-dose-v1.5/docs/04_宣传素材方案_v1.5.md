## 二、视频拍摄全流程

### 2.1 双端录屏方案

#### 桌面端（macOS）
```bash
⌘+⇧+5 → 选择「录制选定部分」→ 框选浏览器手机框 → 录制
```

#### 桌面端（Windows）
```
Win+G 唤起 Xbox Game Bar → 录制
推荐：OBS Studio（更专业，可设置 60fps + 蓝牙麦克风）
```

#### 移动端（iOS）
```
设置 → 控制中心 → 添加「屏幕录制」
从右上角下拉 → 长按录制按钮 → 开启麦克风
```

#### 移动端（Android）
```
下拉通知栏 → 屏幕录制
或装：AZ Screen Recorder
```

### 2.2 双端拍摄机位
```
┌─ 主机位 ─────────────┐
│ 三脚架 · 拍电脑屏     │
│ 距离 50cm · 45°俯角  │
└──────────────────────┘
┌─ 副机位 ─────────────┐
│ 手机支架 · 俯拍手机   │
│ 桌面铺宣纸+茶杯       │
│ 距离 30cm · 90°垂直  │
└──────────────────────┘
```

## 四、剪辑要点

### 4.1 软件推荐
- **入门**：剪映（手机/桌面，免费，模板丰富）
- **进阶**：DaVinci Resolve（免费专业级）
- **专业**：Premiere Pro / Final Cut Pro

### 4.2 关键剪辑节点

| 时间点 | 操作 |
|---|---|
| 0-4s 片头 | 中心放大缩入，淡入淡出 |
| 12-15s 瘦金体特写 | 字体细节慢放 0.8 倍 |
| 18-22s AI 浮层加载 | 4 步文字切换与加载圈同步 |
| 22-32s 双模式切换 | 切换瞬间「叮」音同步 |
| 32-45s 音方笺三段 | 三段 Tab 切换慢速 0.9 倍 |
| 55-58s v1.4/v1.5 字体对比 | 分屏左右对比，停 2 秒 |
| 72-75s 片尾 | 古筝余韵 + Slogan 居中淡出 |

### 4.3 BGM 选择
**推荐免版权古筝 BGM**（YouTube Audio Library 可下）：
- "Bamboo Forest" - 古筝慢板
- "Rain on the Lake" - 钢琴+古筝
- 或用合音页生成「古琴+颂钵+风铃」AI 音轨
