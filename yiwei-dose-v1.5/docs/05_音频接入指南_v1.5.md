# 一味 Dose v1.5 · 音频接入指南

> v1.5 完整保留 v1.3 引入的 **AudioRegistry** 接口，支持注入真实音频文件，无缝替换合成音律。

---

## 🎯 设计目标

```
┌──────────────────────────────────────┐
│           用户点击播放                │
└────────────────┬─────────────────────┘
                 ↓
        AUDIO_REGISTRY.has(id)?
                 ↓
       ┌─────────┴─────────┐
      YES                  NO
       ↓                    ↓
  ┌─────────┐         ┌──────────┐
  │ 加载真实 │         │ 启用合成 │
  │ 音频文件 │  fallback│ 算法引擎 │
  └─────────┘         └──────────┘
```

**优势**：
- ✅ Demo 阶段零依赖（纯合成）
- ✅ 决赛/上线无缝注入真实音频，不重构
- ✅ 单首失败自动 fallback，体验不中断

---

## 🔌 API 文档

```typescript
class AudioRegistry {
  register(instrumentId: string, url: string): void;
  registerBatch(obj: Record<string, string>): void;
  has(instrumentId: string): boolean;
  async load(audioCtx: AudioContext, instrumentId: string): Promise<AudioBuffer|null>;
}
window.AUDIO_REGISTRY  // 已自动挂载
```

## 🚀 快速接入（3 步）

### Step 1：准备音频文件
- 推荐：`.mp3`（兼容性最好）/ `.ogg`（体积小）
- 采样率：≥ 44.1kHz
- 时长建议：30-180 秒
- 体积建议：每首 ≤ 2MB
- 试听音频：3-5 秒，纯净音色

### Step 2：上传到 CDN
| 推荐方案 | 优势 | 注意 |
|---|---|---|
| GitHub raw | 免费、即用 | 单文件 < 25MB |
| jsDelivr | CDN 加速 | 自动镜像 GitHub |
| Cloudflare R2 | 国内访问快 | 需账号 |
| 腾讯云 COS | 国内最快 | 收费 |
| 本地 `/audio/` | 离线可用 | 需打包部署 |

### Step 3：注册 URL
```js
AUDIO_REGISTRY.registerBatch({
  'gz_yunshui':    'https://your-cdn/guzheng_yunshui.mp3',
  'guzheng':       'https://your-cdn/sample_guzheng_3s.mp3',
  // ... 22 件乐器 + 16 首种子曲
});
```
